# 页面布局改版设计 (纯前端)

**项目**: HuHaa-MySkills (Vue 3 + Vite)  
**类型**: 纯前端项目，无后端改造  
**目标**: 重新布局，让搜索/过滤条件完整可见

---

## 📊 问题分析

### 当前问题（从你的截图）

```
用户截图布局：
┌─────────────────────────────────────────────────┐
│ Search Bar (搜索框)                              │
├──────────┬────────────────────────────────────┤
│ Filters  │ List (技能列表)                     │
│ (Sidebar)│                                   │
│          │                                   │
│ 类型     │  ┌────────┐ ┌────────┐            │
│ 来源     │  │ 项目1  │ │ 项目2  │...         │
│ 产品     │  └────────┘ └────────┘            │
│ 点赞     │  ┌────────┐ ┌────────┐            │
│ 指挥     │  │ 项目3  │ │ 项目4  │...         │
│ 获认     │  └────────┘ └────────┘            │

└──────────┴────────────────────────────────────┘

                                   ┌──────────────────┐
                                   │ Detail Panel     │
                                   │ (右侧隐藏或很小) │
                                   └──────────────────┘
```

### 实际问题

❌ **过滤条件显示不全**
   - Sidebar 太窄 (200px 左右)
   - 过滤项超过 6 个时，需要向下滚动才能看到
   - 用户无法一眼看到所有过滤选项

❌ **列表占用空间大**
   - 中间列表占据了大部分宽度
   - 右侧详情面板完全看不见或极小

❌ **响应式不好**
   - Sidebar 固定宽度，适应不了不同屏幕

```
        ┌─────────────────────────────────┐
        │  Detail Panel (浮层/弹窗)        │
        │  点击列表项后显示                │
        │  ┌──────────────────────────┐   │
```

- ✅ 过滤条件全部可见 (240px 足够显示 4-5 个维度)

- ✅ 移动设备也能工作 (Sidebar 可折叠)
- ✅ 不需要后端改造

### 方案 B: **三栏布局 (如果屏幕够大)**

仅在 Desktop 显示，Tablet/Mobile 降级到方案 A。

```
┌─────────────────────────────────────────────────────────────┐
│ Header                                                      │
├────────────┬──────────────────┬──────────────────────────────┤
│ Filters    │ List (60%)       │ Detail (40%)                │
│ (240px)    │                  │                            │
│            │ 卡片视图或列表   │ ┌──────────────────────┐  │
│ 📂 Type    │                  │ │ 项目详情             │  │
│ 📋 Source  │                  │ │                      │  │
│ 🎨 Brand   │ ┌──────────┐    │ │ • 标题               │  │
│ 📁 Category│ │ 项目1   │    │ │ • 标签               │  │
│            │ ├──────────┤    │ │ • 描述               │  │
│            │ │ 项目2   │    │ │ • 元数据             │  │
```

**优点:**
- ✅ 大屏显示更多信息

## 💻 技术实现 (纯 Vue 3)

### 文件结构

```
src/
├── pages/
│   └── index.vue              <- 主页面
├── components/
│   ├── Header.vue             <- 顶部
│   ├── SearchBar.vue          <- 搜索栏
│   ├── FilterSidebar.vue      <- 左侧过滤
│   ├── SkillList.vue          <- 中间列表
│   ├── DetailPanel.vue        <- 右侧浮层详情
│   └── FilterChips.vue        <- 搜索条件标签
├── stores/
│   ├── skills.js              <- 技能数据 (Pinia)
│   └── filters.js             <- 过滤状态 (Pinia)
└── styles/
    └── global.css             <- 全局样式
```

### 数据流

```
FilterSidebar (选择过滤)
    ↓
stores/filters.js (保存过滤状态)
    ↓
SkillList (自动刷新列表)
SearchBar (搜索)
    ↓
stores/skills.js (搜索 + 过滤)
    ↓
ListItem (列表项)
    ↓ (点击)
DetailPanel (显示详情)
```
