┌────────────────────────────────────────────────────────────────────────────────────────┐

│ ┌────────┬────────┬────────┬────────┐                                                  │
│ │ 200px  │ 200px  │ 200px  │ 200px  │   ← 4 个卡片                                    │
│ │ (卡片) │ (卡片) │ (卡片) │ (卡片) │   (总共 800px)                                  │
│ │        │        │        │        │                                                  │
│ │ +16px  │ +16px  │ +16px  │ +16px  │   ← 间距                                       │
│ │ margin │ margin │ margin │ margin │   (总共 64px)                                   │
│ │        │        │        │        │                                                  │
│ │ +32px  │        │        │        │   ← 左右 padding (各 16px)                     │
│ └────────┴────────┴────────┴────────┘                                                  │
│                                                                                         │
│ 计算: 16 + (200×4) + (16×3) + 16 = 16 + 800 + 48 + 16 = 880px (有 800px 可用)       │

│ └────────────────────────┘                                                             │
│                                                                                         │

│ 940px / (140px + 16px) = 6.3 行 → 实际显示 6 行 (840px) + 100px 空余                 │
