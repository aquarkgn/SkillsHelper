## ✨ 新增功能

### 🎯 Hermes 技能集成
```
调用 Hermes 技能 "启动 huhaa" 或 "打开技能中心"
↓
自动检测安装状态和扫描时间
↓
如果今天扫描过 → 直接打开 localhost:11520
如果未扫描或 >1 天 → 提示用户选择是否重新扫描
↓
浏览器打开 HuHaa 网页
```

**Skill 文件**:
- 位置: `~/.hermes/skills/huhaa-myskills/SKILL.md`
- 内容: 完整的安装、启动、配置、故障排查指南
- 功能: 可视化入口，一键启动

### 🔍 前端搜索优化
- ✅ 快捷键支持
  - `/` - 聚焦搜索框
  - `Esc` - 清空搜索
  - `↑ ↓` - 导航列表
  - `Enter` - 打开详情
- ✅ 虚拟滚动 - 流畅处理 800+ 条目
- ✅ 高级搜索 - 支持多条件组合筛选

### 🔧 API 增强
- `/api/search?q=...&source=...&brand=...` - 快速搜索端点
- `/api/state` - 返回扫描时间和统计信息
- SSE 连接稳定性改进

### 📚 文档完善
- ✅ `docs/GUIDE.md` - 5 分钟快速启动指南
- ✅ `docs/RULES.md` - 发布规范（SemVer、Conventional Commits）
- ✅ `.github/ISSUE_TEMPLATE/` - Issue 模板

- ✅ `CHANGELOG.md` - 版本历史
- ✅ `docs/releases/v0.1.x.md` - 版本详细说明

```javascript
// service/scripts/detect-state.mjs
- 检查 npm 全局安装位置
- 读取 sources.yaml 配置
- 检查 cache.json 的扫描时间
- 判断是否需要重新扫描
```
