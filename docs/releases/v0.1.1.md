## 🎯 版本目标

完成 npm 全局安装支持，让 HuHaa-MySkills 可以通过 `npm install -g` 一行命令安装到全局，用户可以在任何目录直接运行 `huhaa-myskills` 命令。

## 📋 版本主题

**npm 全局安装支持** - 从需要 `git clone` 和 `npm install` 的本地项目，演进到可以全局安装使用的命令行工具。

## 🔧 技术改进

### npm 打包配置
```json
{
  "bin": {
    "huhaa-myskills": "service/bin/huhaa-myskills.mjs"
  },
  "files": [
    "service/bin/",
    "service/config/",
    "service/packages/",
    "service/scripts/build-web.mjs",
    "docs/",
    "README.md",
    "LICENSE"
  ]
}
```

### Web 资产分发
- 集成了预构建的 Vue 前端资产
- 减少了 npm install 后的构建时间
- 提高了全局安装的可靠性

### 依赖优化
- 仅在 npm 安装时安装必要的依赖
- Runtime 依赖和开发依赖清晰分离

## 🐛 Bug 修复

### 修复列表

| Issue | 问题 | 解决方案 |
|-------|------|---------|
| #1 | npm bin 路径前缀 `./ ` 在某些环境下不兼容 | 移除前缀，改用相对路径 |
| #2 | Git install 时 web 资产丢失 | 将预构建的 dist 目录包含在 npm 包中 |
| #3 | 全局安装后 node_modules 的依赖解析问题 | 优化了 workspaces 的依赖链接 |
| #4 | npm install 期间会执行 prepare 脚本导致重复构建 | 修复了脚本执行流程 |

## 🚀 安装指南

### 方式 1：项目克隆（开发）
```bash
git clone https://github.com/aquarkgn/HuHaa-MySkills.git
cd HuHaa-MySkills
npm install
npm start
```

### 方式 2：全局 npm 安装（推荐日常使用）
```bash
# 安装最新版本
npm install -g github:aquarkgn/HuHaa-MySkills

# 或指定版本
npm install -g github:aquarkgn/HuHaa-MySkills@v0.1.1

# 启动
huhaa-myskills start
```

### 升级
```bash
npm update -g github:aquarkgn/HuHaa-MySkills
```
