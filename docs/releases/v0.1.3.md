### 🔧 编辑器检测增强

**平台特定检测**
- macOS: `~/Library/Application Support/{Editor}`
- Linux: `~/.config/{editor}`

**多版本支持**
- Sublime Text 3 & 4
- VS Code & VS Code Insiders
- JetBrains 系列 IDE

**智能配置路径**
```bash
# 示例
Cursor          → ~/.cursorrules
VS Code         → ~/.config/Code/User/snippets
Neovim          → ~/.config/nvim/plugin
Vim             → ~/.vim/plugin
Sublime Text    → ~/.config/sublime-text-3/Packages/User
JetBrains       → ~/.config/JetBrains/*/
```

## 🔧 技术改进

### 脚本重构

**detect_editors() 函数**
- 优化循环结构，避免重复
- 使用 local os 变量减少系统调用
- 支持更多编辑器的灵活扩展

**同步函数统一**
- 所有 sync_to_* 函数遵循相同签名
- 智能目录创建（mkdir -p）
- 统一的日志输出

### 编辑器配置模式

```bash
# 新增 sync 函数示例
sync_to_zed() {
  local editor_path="$1" root="$2"
  mkdir -p "$editor_path"
  log_success "Zed: 已同步"
}

sync_to_jetbrains() {
  local editor_path="$1" root="$2"
  mkdir -p "$editor_path"
  log_success "JetBrains IDEs: 已同步"
}
```
