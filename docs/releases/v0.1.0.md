## 🎯 版本目标

完成 P0-P7 完整实现（8 小时净工作量），交付一个可用的本地技能聚合浏览器，实现多源技能的统一查找和复制。

## 📋 版本主题

**本地技能聚合中心** - 将散落在 Hermes / Claude Code / Cursor / MCP 等多个工具中的技能统一汇聚到一个本地浏览器，提供搜索、筛选、复制等快速操作。

---

## ✨ 新增功能

### 核心功能
- 🎯 **多源聚合** - 聚合 Hermes、Claude Code、Cursor Rules、MCP Config、Project Runbook 等 5+ 个源
- 🔍 **全文搜索** - 基于 Fuse.js 的前端全文搜索，延迟 < 5ms
- 📂 **智能分组** - 支持按源、分类、品牌、产品四种维度分组浏览
- 📋 **详情页** - 查看技能的完整信息、参数、使用说明、相关链接
- 📋 **一键复制** - 复制路径、复制原文、复制 frontmatter
- 🔗 **编辑器跳转** - 点击"在 Cursor 中打开"直接打开文件
- 🔄 **热更新** - 修改技能文件后 < 1s 自动刷新页面（chokidar watch）

### 技术架构
- **前端**: Vite 5 + Vue 3 + TypeScript + Pinia + Vuetify
- **后端**: Fastify 5 + Node.js 20+
- **扫描**: 多源适配器 + 统一 IR 数据结构
- **通信**: SSE (Server-Sent Events) 实时推送

### 支持的源

| 源 | 路径 | 类型 | 说明 |
|---|------|------|------|
| Hermes | `~/.hermes/skills/*/SKILL.md` | skill | Hermes 官方技能库 |
| Claude Code | `~/.claude/skills/*/SKILL.md` | skill | Claude Code 技能 |
| MCP Config | `~/.cursor/mcp.json` 等 | mcp | MCP 服务器定义 |
| Cursor Rules | `.cursorrules` / `.cursor/rules/` | agent-rule | Cursor 规则 |
| Project Runbook | `docs/RUNBOOK-*.md` | runbook | 项目文档 |

## 🐛 已知限制

### 功能限制
- ❌ 不支持在网页端编辑技能（只读浏览）
- ❌ 不支持跨机器同步（本地工具）
- ❌ 不支持直接执行技能（复制后由编辑器执行）
- ❌ Windows / Linux 尽量兼容但不是一等公民

### 平台兼容性
- ✅ macOS - 完全支持
- ⚠️ Linux - 功能完整，部分工具集成需要调整
- ⚠️ Windows - 功能完整，路径和剪贴板需要调整

## 🚀 安装使用

### 快速开始
```bash
git clone https://github.com/aquarkgn/HuHaa-MySkills.git
cd HuHaa-MySkills
npm install
npm start
# 浏览器自动打开 http://localhost:11520
```

### 常用命令
```bash
npm start       # 启动并打开浏览器
npm run scan    # 仅扫描，输出 IR JSON
npm run stats   # 统计信息
npm run dev     # 开发模式
npm test        # 运行测试
npm run verify  # 完整性检查
```

## 🎨 使用场景

### 场景 1：快速查找 Hermes 技能
1. 浏览器打开 localhost:11520
2. 搜索框输入关键词
3. 找到目标技能
4. 点击"复制原文"
5. 粘贴到 Cursor 上下文

### 场景 2：浏览所有 MCP 服务
1. 分组切换为"source"
2. 选择"mcp-config"
3. 查看所有 MCP 服务定义
4. 复制工具定义到 Cursor

### 场景 3：快速查看项目 Runbook
1. 分组切换为"kind"
2. 选择"runbook"
3. 查看所有项目文档
4. 点击"在 Cursor 中打开"

## 🔧 配置文件

### sources.yaml

首次启动自动生成在用户数据目录：
```yaml
sources:
  hermes:
    enabled: true
    roots:
      - ~/.hermes/skills
  
  claude-code:
    enabled: true
    roots:
      - ~/.claude/skills

  mcp-config:
    enabled: true
    files:
      - ~/.cursor/mcp.json

limits:
  maxFiles: 5000
  maxFileBytes: 1048576
```

## 🔐 安全设计

- ✅ Localhost only - 绑定 127.0.0.1，不支持远程访问
- ✅ 路径白名单 - `/api/open` 和 `/api/copy` 都有白名单验证
- ✅ 无上传下载 - 纯只读浏览，不写入用户文件
- ✅ 无网络通信 - 完全本地工作，无任何远程请求
