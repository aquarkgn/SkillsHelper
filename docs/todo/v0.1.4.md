# v0.1.4：分组与筛选增强

## 目标
支持四种分组方式切换，提供高级筛选面板，改进导航体验。

## 关键任务

### 前端
- [ ] 实现分组切换按钮组：source / category / brand / product
  - [ ] 每种分组方式下树状或卡片布局
  - [ ] 分组数据缓存，切换不重新计算
- [ ] 新增筛选面板：
  - [ ] 多选源、分类、品牌
  - [ ] 文本输入框与 Fuse.js 搜索组合
  - [ ] 筛选结果实时更新
- [ ] 改进详情页导航：prev/next 按钮在同分组内切换

### UI
- [ ] 面包屑导航显示当前分组路径
- [ ] 未匹配项提示友好

### 测试
- [ ] 分组计算正确性 > 90%
- [ ] 筛选组合场景覆盖

## 验收标准
- 四种分组方式可切换
- 筛选组合生效
- 面包屑导航准确

## 预期周期
1.5 周

## 不包含
- 不做收藏/书签功能
- 不做分组嵌套（固定四层）
