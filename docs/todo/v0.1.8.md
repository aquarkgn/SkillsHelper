# v0.1.8：开发者工具与调试

## 目标
增强 CLI 和调试能力，方便用户自定义和二次开发。

## 关键任务

### CLI 增强
- [ ] `huhaa-myskills` 新命令：
  - [ ] `huhaa-myskills doctor` — 诊断配置和数据问题
  - [ ] `huhaa-myskills validate-config` — 验证 sources.yaml
  - [ ] `huhaa-myskills export <format>` — 导出 IR
- [ ] 支持配置文件路径覆盖：`HUHAA_CONFIG` env
- [ ] 支持日志级别：`LOG_LEVEL=debug npm start`

### 调试工具
- [ ] 前端 DevTools：集成 Pinia inspector 快捷启用
- [ ] API 响应时间统计：`/api/stats?verbose=1`
- [ ] 扫描性能分析：按适配器统计耗时

### 文档
- [ ] 编写适配器开发指南（如何新增源类型）
- [ ] 配置参考文档
- [ ] 常见问题排查指南

## 验收标准
- `doctor` 命令能检测常见问题
- 调试日志清晰可读
- 开发指南可用于第三方扩展
