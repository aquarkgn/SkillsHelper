# v0.1.2：Hermes 技能集成 + 搜索优化

## 目标
创建 Hermes skill 作为 HuHaa 的一等入口，实现一键启动、智能状态检测、热更新查看。同时完善项目文档规范，参考 GitHub 官方标准。

### 🔍 启动智能检测
- [ ] 实现状态检测脚本：`service/scripts/detect-state.mjs`
