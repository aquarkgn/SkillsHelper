# v0.1.2：搜索与交互优化

## 目标
提升搜索体验和 UI 交互流畅度，修复 v0.1.1 反馈的交互问题。

## 关键任务

### 前端
- [ ] 改进搜索框 UX：支持快捷键 `/` 聚焦搜索、支持 `Escape` 清空
- [ ] 优化列表虚拟滚动，处理 800+ 条目性能
- [ ] 改进详情页加载状态：skeleton 或 loading spinner
- [ ] 支持键盘导航（上/下箭头切换列表，Enter 打开详情）

### Server
- [ ] 添加 `/api/search?q=...&filters=...` 快速搜索端点，支持过滤语法
- [ ] 优化 SSE 连接稳定性，处理长连接断开重连
- [ ] 改进错误响应，返回更清晰的错误信息

### 扫描器
- [ ] 增强 frontmatter 解析容错，支持更多 YAML 变体
- [ ] 改进 brand 和 category 推断逻辑
- [ ] 添加扫描统计信息（成功/失败/跳过）

## 测试
- [ ] 搜索功能覆盖率 > 85%
- [ ] UI 响应时间 < 100ms（800 条目）

## 验收标准
- 搜索延迟 < 5ms（Fuse.js）
- 详情页加载不卡顿
- 键盘导航可用

## 预期周期
1 周

## 不包含
- 不做高级搜索语法
- 不做搜索历史记录
