# v0.1.3：Obsidian Vault 与 Claude Markdown 支持

## 目标
聚合 Obsidian vault 中标记 `#skill` 的笔记，支持 CLAUDE.md 全局指令文档。

## 关键任务

- [ ] 实现 `obsidian.adapter.mjs`：扫描 vault 中的 frontmatter tag
  - [ ] 支持 `#skill` 和 `#plugin` 标签
  - [ ] 提取 frontmatter 的 name / description / tags
  - [ ] 从 note 前 50 行提取 preview
- [ ] 实现 `claude-md.adapter.mjs`：项目根和 `~/.claude/CLAUDE.md`

  ```yaml
    enabled: false  # 用户手动启用
    vaultPath: ~/Library/Mobile Documents/com~apple~CloudDocs/Obsidian
    tags: [skill, plugin, mcp]
  ```
