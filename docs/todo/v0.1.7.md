# v0.1.7：跨平台完整支持

## 目标
确保 Windows 和 Linux 一等公民支持，改进平台特定工具集成。

## 关键任务

### Windows 支持
- [ ] 剪贴板：用 powershell / `clip.exe` 替代 `pbcopy`
- [ ] 文件打开：支持 `explorer.exe` / `start`
- [ ] 编辑器启动：支持 VSCode / Cursor (Windows)
- [ ] 路径规范化：统一处理 `\` 和 `/`

### Linux 支持
- [ ] 剪贴板：支持 `xclip` / `xsel` / `wl-copy`
- [ ] 文件打开：支持 `xdg-open`
- [ ] 编辑器启动：支持 code / cursor / vim
- [ ] Wayland 兼容性

### 测试
- [ ] Windows / Linux 上运行 `npm test` 全绿
- [ ] 路径穿越防护在各平台有效

### 文档
- [ ] 更新 README 平台支持说明
- [ ] 添加 Windows / Linux 快速启动指南

## 验收标准
- Windows 上 `npm start` 完全可用
- Linux 上 `npm start` 完全可用
- 三个平台功能等价

## 预期周期
1.5 周

## 不包含
- 不做移动端
- 不做容器内环境（Docker）
