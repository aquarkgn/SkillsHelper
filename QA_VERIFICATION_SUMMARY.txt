🔧 最新修复（提交 a6d17bd）

问题: Vue @click 事件绑定可能因框架问题无法正确触发

解决方案: 使用原生 JavaScript 事件监听 + data 属性标记

代码改动:

  1. HTML 改动 (第 307 行):
     - 从: <button @click="closeDetail()">✕</button>
     + 到: <button data-action="close-modal">✕</button>

  2. JavaScript 改动 (第 31-42 行):
     - 在 onMounted 中添加原生事件监听
     - document.addEventListener('click', (e) => {
         if (e.target.closest('[data-action="close-modal"]')) {
           console.log('Close modal clicked');
           closeDetail();
         }
       });

优点:
  ✓ 绕过 Vue 事件系统可能的问题
  ✓ 直接访问 DOM，可靠性更高
  ✓ 添加 console.log 便于调试和验证
  ✓ 使用 closest() 安全地查找目标元素
