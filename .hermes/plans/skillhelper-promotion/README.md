# SkillHelper 抖音推广资料库

这里是 **SkillHelper（呼哈哈-技能助手）** 当前唯一的推广运营工作区。现阶段只推广抖音，不维护其它平台计划。

## 品牌口径

- 对外名称：**SkillHelper**。
- 中文说明：**呼哈哈-技能助手**。
- 一句话：**把散落在本机、编辑器、插件与 MCP 配置中的 AI 能力，整理成一个可搜索、可理解、可持续维护的工作台。**
- 技术身份：npm 包名和 CLI 命令均为 `skillhelper`。
- 内容原则：真实录屏、真实命令、真实数据优先；AI 只用于脚本、配音、字幕、封面和转场包装，并按抖音规则标识。

## 文件导航

| 文件/目录 | 单一职责 | 更新时机 |
| --- | --- | --- |
| [镜头资产管理规范.md](./镜头资产管理规范.md) | 镜头管理、状态流转、命名和存储边界 | 工作流变化时 |

| [campaigns/2026-Q3-抖音推广计划.md](./campaigns/2026-Q3-抖音推广计划.md) | 抖音定位、内容节奏、实施计划与复盘口径 | 每周复盘后 |

## 目录边界

```text

├── assets/      # YAML 清单、字幕稿、封面文案
├── campaigns/   # 抖音计划与发布记录
├── metrics/     # 抖音指标快照
├── videos/      # 视频资产工作区；大视频默认不提交

```
