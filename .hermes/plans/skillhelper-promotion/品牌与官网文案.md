## 官网定稿文案

**眉标**：SkillHelper · 本地 AI 工作流能力中枢

**标题**：AI 技能、插件、MCP 配置散落各处？把它们变成你的能力工作台。

**说明**：扫描本机技能、插件、MCP 配置与项目运行手册，统一查看、搜索、分类、筛选；需要时查看中文对照，命令与代码保持原样。

**主 CTA**：GitHub 仓库

**次 CTA**：快速开始

```bash
npm install -g skillhelper@latest
skillhelper start
```

## 抖音固定口径

- 开场：AI 技能装多了，反而找不到？
- 定位：SkillHelper 把本机 AI 能力整理成一个可搜索的工作台。
- 证据：真实命令、真实面板、真实搜索和筛选。
- 边界：扫描只读；敏感配置展示前脱敏；翻译功能可能联网。
- CTA：评论“技能助手”，我发安装入口。

## 禁用表达

- 不说“所有数据永不出本机”：用户触发翻译时会调用翻译服务。
- 不把“本机扫描到的数量”写成产品恒定能力。
- 不把未实时验证的版本、release 数、Star/Fork 数放入官网和封面。
- 不称为“聊天助手”或“万能 AI 平台”；聚焦能力发现、整理和理解。
- 不再使用旧英文名或多平台推广口径。
