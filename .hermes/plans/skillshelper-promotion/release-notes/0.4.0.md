# SkillsHelper 0.4.0 发布说明

发布日期：2026-07-12

发布状态：已发布到 npm，`latest` 指向 `0.4.0`。

registry 记录时间：2026-07-12 09:54:30 UTC。

## 版本定位

0.4.0 是呼哈哈-技能助手完成品牌迁移后的首个发布准备版本。中文对外名称统一为「呼哈哈-技能助手」，英文与技术身份统一为 `SkillsHelper`，npm 包名和 CLI 入口统一为 `skillshelper`。

## 重要变更

- 项目包名改为 `skillshelper`。
- CLI 入口改为 `skillshelper`。
- 默认配置目录改为 `~/.config/skillshelper`。
- 环境变量前缀改为 `SKILLSHELPER_`。
- 官网、工作台、favicon、manifest 和主品牌 Logo 已迁移到新品牌。
- 推广计划、镜头清单和视频素材索引统一迁移到 `.hermes/plans/skillshelper-promotion/`。
- 扫描器新增 Claude Agents、Claude Plugins、Claude MCP、Codex MCP、Codex Plugin、Hermes Skills、Hermes Plugins 等来源识别与前端补充展示。
- Hermes 图标增加本地内置资源，减少依赖远程图标获取。

## 不兼容变更

以下旧入口不再作为兼容目标：

- 旧命令：`huhaa-myskills`
- 旧配置目录：`~/.config/huhaa-myskills`
- 旧环境变量前缀：`HUHAA_`

迁移后请使用：

```bash
skillshelper start
skillshelper init
```

配置文件位置：

```text
~/.config/skillshelper/sources.yaml
```

环境变量示例：

```bash
export SKILLSHELPER_HOME="$HOME/.config/skillshelper"
```

## 升级建议

1. 安装或更新 npm 包：

   ```bash
   npm install -g skillshelper@latest
   ```

2. 初始化新配置目录：

   ```bash
   skillshelper init
   ```

3. 如需迁移旧配置，手动将旧 `sources.yaml` 中仍有效的来源复制到 `~/.config/skillshelper/sources.yaml`。

4. 启动本地工作台验证：

   ```bash
   skillshelper start
   ```
