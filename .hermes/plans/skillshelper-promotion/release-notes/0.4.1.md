# SkillsHelper 0.4.1 发布说明

发布日期：2026-07-12

发布状态：已发布到 npm，`latest` 指向 `0.4.1`。

registry 记录时间：2026-07-12 10:27:24 UTC。

## 版本定位

0.4.1 是 0.4.0 之后的 patch 版本，重点修复 CLI 机器可读输出的稳定性，确保后续脚本、镜头工具和录屏演示可以直接消费 `scan --json` 的 stdout。

## 修复内容

- `skillshelper scan --json` 的 stdout 现在只输出 JSON。
- 扫描进度日志改为输出到 stderr。
- 增加 CLI 级回归测试，确保 stdout 可以直接 `JSON.parse`。
- 更新首发镜头 SH-003 的录制备注，明确 stdout/stderr 分流后的演示方式。

## 升级建议

```bash
npm install -g skillshelper@latest
skillshelper --version
skillshelper scan --json > skills.json
```

验收标准：

- `skillshelper --version` 输出 `skillshelper v0.4.1`。
- `skills.json` 可直接被 JSON 解析。
- 扫描进度仍可在终端 stderr 中看到。

在仓库外使用全新 npm cache 完成干净安装验收，避免命中本地旧包缓存。

```bash
npm install --prefix "$prefix" --cache "$cache" --prefer-online --omit=dev skillshelper@latest
"$prefix/node_modules/.bin/skillshelper" --version
```
